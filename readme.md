# BE NATIVE

## Requirements

- 1: `PHP 8.0 or Higher`
- 2: `PHP PDO MSQL`
- 3: `MSQL`

## How to use

- 1: `Import be.sql to Mysql`
- 2: `Import Postman Collection (if needed)`
- 3: `.htaccess`

The `.htaccess` redirects all requests to `index.php`.

## Endpoints
- POST /login
- POST /register
- POST /user
- GET /userById?=1
- GET /user
- DELETE /user?id=1
- PUT /uploadimage
